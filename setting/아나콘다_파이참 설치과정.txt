-- 파이참 환경 설정 -- 
1) 작업 폴더 생성 
   예) pyclass
2) 프로젝트 설정 
   프로그램 첫 실행시 표시되는 창에서 
   [New Project] 
   - 새로 프로젝트 생성시에는 [File]-[New project]  
   - 작업폴더 설정 
   - 프로젝트 수정은 [File]-[Settings]
     [Project] 카테고리에서 변경 
3) 파이썬 파일 만들어서 확인 (*.py)
    helloworld.py 
  
    # 파이썬 한줄 주석 ( Ctrl+/ )
    print("Hello World")

    # 실행 
    shift + F10 
    [Run]-[Run ~]

4) 파이썬 에디터 기본 글꼴 
    [File]-[Settings]-[Editor]-[Font]
